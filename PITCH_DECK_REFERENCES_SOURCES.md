### How We Calculated CAC

```

- Phase 1 (M1-6): $50K ‚Üí 650 users = $77 CAC
- Phase 2 (M7-12): $150K ‚Üí 1,500 users = $100 CAC
- Phase 3 (M13-18): $150K ‚Üí 1,000 users = $150 CAC

- 30% of users come via referrals/PR (essentially $0 CAC)

- So we use $111 blended ‚úÖ
```

### How We Calculated LTV

```
INPUTS:
- Average revenue per user (paid only): $62.58/month

- Gross margin: 78%

FORMULA:
LTV = ARPU √ó Lifetime √ó Gross Margin

LTV = $976

OPTIMISTIC (30 months, lower churn):
LTV = $62.58 √ó 30 √ó 0.78 = $1,464

```
